# 📌 About The Project

**Profile Card UI** is a responsive frontend project that includes:

* A modern profile card homepage
* Elegant login page
* Stylish sign up page

The main highlight of this project is its:

* Premium Mesh Glow Gradient Background
* Smooth UI Design
* Clean Typography
* Responsive Layout

The entire project is built only using **HTML5** and **CSS3** without any frameworks.
